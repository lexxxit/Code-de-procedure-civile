# Article ANNEXE, art. 21

Les intéressés demeurant à l'étranger sont tenus de désigner, dans le département où le notaire est établi, un fondé de
pouvoir chargé de recevoir les notifications, faute de quoi celles-ci seront acheminées par la voie postale.

Les notifications aux absents se font entre les mains de leur représentant ou curateur.

Les notifications sont réglées, au surplus, par les dispositions du code de procédure civile.
